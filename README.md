# カレンダーアプリ
カレンダーにスケジュールを追加できます。
スケジュールをタップするとその詳細を見ることができます。
スケジュールは削除、編集可能です。
データの永続化はしておりません。

## 機能一覧
- カレンダー表示機能
- スケジュール投稿・削除・編集機能
- モーダル機能
